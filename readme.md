#  Resume AI Screening System

An intelligent AI-powered resume screening platform that automates candidate evaluation using Machine Learning, NLP, and data visualization.

---

##  Features

-  Resume Classification (Domain Prediction)
-  Job Description Matching
-  Skill Detection using NLP + Fuzzy Matching
-  HR Screening Dashboard
-  Multi-Resume Comparison
-  Interactive Visual Analytics
-  Automated PDF Report Generation

---

##  Tech Stack

- Python
- Scikit-learn (TF-IDF + Naive Bayes)
- Gradio (UI)
- Plotly (Visualization)
- PyPDF2 (Resume Parsing)
- ReportLab (PDF Export)
